# Research

Shared knowledge base for Rolf, Annie, and Claude Code.

When you find something relevant — a study, a paper, a competitor
feature, a user insight — add it here so the whole team benefits.

## Sub-folders

- `papers/` — Academic studies, clinical research, accessibility literature.
  Add a short `[filename].md` alongside each PDF summarizing the key
  takeaways and why it's relevant to MedEase.

- `competitive/` — Notes on other tools in the accessibility and
  healthcare assistant space. One markdown file per tool.

- `user-insights/` — Anything learned from real or potential users:
  interview notes, survey results, informal feedback.

## Convention

Every file should start with a date and source:

```
> Added: YYYY-MM-DD
> Source: [URL or citation]
> Added by: [Rolf / Annie / Claude]
```
