### Demo UI/UX Overhaul (`feat/demo-uiux` → dev → main)

All demo pages were previously overflowing their viewport. This was a systematic fix:

**Layout architecture:**
- Outer container: `height: calc(100vh - 64px), overflow: hidden`
- Card: `flex: 1, minHeight: 0, overflow: hidden, display: flex, flexDirection: column`
- Content regions: `flex: 1, minHeight: 0, overflowY: auto`
- See [[adr-004-demo-uiux-layout|ADR-004]] for full rationale.

**`DemoSectionNav` shared component** — extracted from 5+ copy-paste copies. Centralizes active-state logic and nav item list.

**`/home` improvements:**
- Care stages: compact pill chips (filled = active, outlined = inactive) instead of spread text buttons with a heart dot
- Recommendation panel: collapsed by default, expand with one click
- Chat: fills remaining height, scrolls internally as messages grow
- Chat layout: replaced MUI Grid (broken height inheritance) with plain flex Box

**Community:**
- Hub cards redesigned as horizontal rows (icon + title/desc + arrow) — 4 cards fit comfortably in viewport
- `CommunityLayout` viewport-constrained, uses `DemoSectionNav`

**Notes:**
- Tab labels: MUI default blue fixed with `textColor="inherit"` on `Tabs`
- Week view: trimmed from 24 hours to 7am–10pm (540px vs 1152px), wrapped in `maxHeight: 420` scroll container
- Month view: cells capped at 36px, tighter header, accent weight on days with notes

**Survey shell:**
- `SurveyShell` outer: `height: calc(100vh - 64px)`, card `maxHeight: calc(100vh - 128px)` with internal scroll
